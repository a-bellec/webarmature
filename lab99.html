Ma première carte ...

Pour commencer, nous allons créer une simple visualisation à partir d'un fond de carte Open Street Map.

Le résultat doit s'approcher de cet exemple


Fichiers Leaflet : CSS & JS

L'utilisation de Leaflet nécessite de faire appel à des fichiers CSS et JS dans le fichier HTML.
Dans les exemples suivants, nous allons appeler directement via internet les fichiers qu'ils nous faut. Pour cela, il va falloir insérer ce code dans la section <head> de la page HTML.


<link rel="stylesheet" href="http://cdn.leafletjs.com/leaflet-0.7.3/leaflet.css"/><script src="http://cdn.leafletjs.com/leaflet-0.7.3/leaflet.js"></script>

Ajout de l'élément Div et stylage

Leaflet a besoin d'un élément <div/> contenant la carte. Cet élément du <div/> contient les informations relatives à la taille de la carte.
Créez un élément <div/> avec l'id "map" en ajoutant ce qui suit :

<div id="map"></div>

Pour créer une carte de 960px de largeur et 500px de hauteur, ajoutez ce code dans le fichier css, ou entre balises <style> dans la section <head> de la page HTML.

#map {
width: 960px;
height: 500px;
}

A partir de maintenant, les bases sont présentes pour créer la carte. Il ne manque juste que quelques lignes de JavaScript.



Appeler le fond de carte OSM


Insérez le code suivant entre les balises <script> du fichier HTML.


<script>
 var map = L.map('map',{
 center: [45.7604276, 4.8335709],
 zoom: 13
 });
 L.tileLayer('http://{s}.tile.osm.org/{z}/{x}/{y}.png', {
 attribution: '© <a href="http://osm.org/copyright">OpenStreetMap</a> contributors'
 }).addTo(map);
</script>


La variable center indique les coordonnés du centre de la carte, et la variable zoom gerè le niveau de zoom par défaut.
La ligne attribution permet de définir la légende de la carte qui s'affichera, par défaut, en bas à droite.

Quelques précisions avant de commencer
Le but de cet article est de présenter une manière d'afficher un jeu de données issu de relevés GPS. Pour pouvoir utiliser vos données sans trop de difficultés, il vous faudra :
  - Importer les données GPS dans un SIG
  - Veiller à renseigner le système de coordonnées au format latitude/longitude
  - Exporter les données au format JSON
  - Mettre votre jeu de donnée JSON dans un des répertoires de travail de votre projet.

L.geoJson(data_point,{
      pointToLayer: function(feature,latlng){
        return L.marker(latlng,{icon: styleIcon});
      }

Pour définir l'affichage de la popup.
L'ajout de cette fonction est relativement simple, il s'agira d'ajouter un argument bindPopup pour permettre l'intereaction avec les points.
A partir de la table attributaire de la donnée nous souhaitons afficher le nom de site (SiteName) et ses coordonnées x ; y.
Pour cela :

marker.bindPopup(
    'Site name : ' + feature.properties.SiteName + '' + 
    'Long. : ' + feature.properties.XCOORD + '' + 
    'Lat. : ' + feature.properties.YCOORD
    );

Les paramètres renseignés dans marker.bindPopup permettent d'afficher sur une 1ère ligne le nom de site, sur une seconde la longitude et sur la derniè la latitude du point.
La première partie dans les '' permet de définir ce qui va s'afficher dans la popup (ici ; SiteName). Le "+" indique qu'une information arrive à la suite. feature.properties.SiteName quand à lui, retourne la valeur correspondante au point sélectionné.

Ensuite il ne reste plus qu'à permettre l'affichage de la popup de cette manière :

return marker;

La partie du code adaptée doit maintenant ressembler à ceci :

pointToLayer: function(feature,latlng){
  var marker = L.marker(latlng,{icon: styleIcon});
  marker.bindPopup('Site name : ' + feature.properties.SiteName + '' + 
      'Long. : ' + feature.properties.XCOORD + '' + 
      'Lat. : ' + feature.properties.YCOORD);
  return marker;
}
