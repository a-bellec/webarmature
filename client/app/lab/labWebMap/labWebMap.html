Fichiers Leaflet : CSS & JS

L'utilisation de Leaflet nécessite de faire appel à des fichiers CSS et JS dans le fichier HTML.

Dans les exemples suivants, nous allons appeler directement via internet les fichiers qu'ils nous faut. Pour cela, il va falloir insérer ce code dans la section <head> de la page HTML. 

<link rel="stylesheet" href="http://cdn.leafletjs.com/leaflet-0.7.3/leaflet.css"/><script src="http://cdn.leafletjs.com/leaflet-0.7.3/leaflet.js"></script>

Ajout de l'élément Div et stylage

Leaflet a besoin d'un élément <div/> contenant la carte. Cet élément du <div/> contient les informations relatives à la taille de la carte.

Créez un élément <div/> avec l'id "map" en ajoutant ce qui suit : 

<div id="map"></div>

Pour créer une carte de 960px de largeur et 500px de hauteur, ajoutez ce code dans le fichier css, ou entre balises <style> dans la section <head> de la page HTML. 

#map {
width: 960px;
height: 500px;
}

À partir de maintenant, les bases sont présentes pour créer la carte. Il ne manque juste que quelques lignes de JavaScript.


Appeler le fond de carte OSM

Insérez le code suivant entre les balises <script> du fichier HTML. 

<script>
 var map = L.map('map',{
 center: [45.7604276, 4.8335709],
 zoom: 13
 });
 L.tileLayer('http://{s}.tile.osm.org/{z}/{x}/{y}.png', {
 attribution: '© <a href="http://osm.org/copyright">OpenStreetMap</a> contributors'
 }).addTo(map);
</script>

La variable center indique les coordonnés du centre de la carte, et la variable zoom gère le niveau de zoom par défaut.

La ligne attribution permet de définir la légende de la carte qui s'affichera, par défaut, en bas à droite.


Maintenant votre page HTML doit ressembler à ceci

<html>
 <head>
  <title>Ma première carte</title>
  <link rel="stylesheet" href="http://cdn.leafletjs.com/leaflet-0.7.3/leaflet.css" />
  <script src="http://cdn.leafletjs.com/leaflet-0.7.3/leaflet.js"></script>
  <style>
   #map {
    width: 960px;
    height:500px;
   }
  </style>
 </head>

    attribution: '© <a href="http://osm.org/copyright">OpenStreetMap</a> contributors'
    }).addTo(map);
  </script>
 </body>
</html>

Gestion des layers

La première carte permet d'afficher un fond OpenStreetMap, nous allons voir ici comment gérer plusieurs fonds de carte et comment permettre une interaction entre ces fonds. Nous utilisons 3 fonds de carte.

Le résultat doit s'approcher de cet exemple :

La structure générale du code reste semblable, nous allons juste ajouter quelques éléments en créant plusieurs variables.


Ajout de la variable référence (link)

Nous ajoutons une variable permettant de gérer les liens internet de nos sources. 

var osmLink = 'OpenStreetMap',
thunLink = 'Thunderforest',
esriLink = 'Esri'

Ajout de la variable adresse (Url) et attribution (Attrib)

Nous ajoutons une variable permettant de gérer les liens d'appel de la source et la manière dont va s'afficher la source en bas à droite de la carte. 

var osmUrl = 'http://{s}.tiles.wmflabs.org/bw-mapnik/{z}/{x}/{y}.png',
osmAttrib = '© ' + osmLink + ' Contributors',
landUrl = 'http://{s}.tile.thunderforest.com/landscape/{z}/{x}/{y}.png',
thunAttrib = '© '+thunLink,
esriUrl = 'http://server.arcgisonline.com/ArcGIS/rest/services/World_Imagery/MapServer/tile/{z}/{y}/{x}',
esriAttrib = '© '+esriLink;

Ajout de la variable carte

Nous ajoutons une variable permettant de gérer ensemble les fonds et les légendes. 

var osmMap = L.tileLayer(osmUrl, {attribution: osmAttrib}),
landMap = L.tileLayer(landUrl, {attribution: thunAttrib}),
esriMap = L.tileLayer(esriUrl, {attribution: esriAttrib});

Ajout de la variable base

Nous ajoutons une variable permettant de gérer la couche à afficher au chargement de la page, le centre de la carte et le niveau de zoom. 

var map = L.map('map', {
layers: [osmMap]
})
.setView([45.7604276, 4.8335709], 18);

Ajout de la variable nom

Nous ajoutons une variable permettant de gérer l'affichage du nom des couches.

var baseLayers = {
"OSM B&W": osmMap,
"Landscape": landMap,
"Aerial": esriMap
};

Ajout de la fonction contrôle

Nous ajoutons une fonction permettant de gérer le contrôle des couches. 

L.control.layers(baseLayers).addTo(map);

Maintenant votre page HTML doit ressembler à ceci 

<!DOCTYPE html>
<html>
 <head>

  <link rel="stylesheet" href="http://cdn.leafletjs.com/leaflet-0.7.3/leaflet.css" />

  <style>
   #map {
    width: 960px;
    height:500px;
   }
  </style>
 </head>

<body>
  <div id="map"></div>
  <script>
   var osmLink = 'OpenStreetMap',
    thunLink = 'Thunderforest',
    esriLink = 'Esri';
   var osmUrl = 'http://{s}.tiles.wmflabs.org/bw-mapnik/{z}/{x}/{y}.png',
    osmAttrib = '© ' + osmLink + ' Contributors',
    landUrl = 'http://{s}.tile.thunderforest.com/landscape/{z}/{x}/{y}.png',
    thunAttrib = '© '+thunLink,
    esriUrl = 'http://server.arcgisonline.com/ArcGIS/rest/services/World_Imagery/MapServer/tile/{z}/{y}/{x}',
    esriAttrib = '© '+esriLink;
   var osmMap = L.tileLayer(osmUrl, {attribution: osmAttrib}),
    landMap = L.tileLayer(landUrl, {attribution: thunAttrib}),
    esriMap = L.tileLayer(esriUrl, {attribution: esriAttrib});
   var map = L.map('map', {
    layers: [osmMap]
     })

    "OSM B&W": osmMap,

Afficher mes données

Dans cette section nous allons voir, pas à pas, comment insérer nos données sur la carte.

Le résultat doit s'approcher de cet exemple :


Il est possible que vous souhaitiez afficher votre propres données.L'exemple présent ici va nous permettre d'afficher un fichier de point au format vectoriel directement sur la carte.

Nous utilisons un fichier au format GeoJSON issu d'un Shapefile convertit avec le logiciel QGIS. Plus d'informations sur le format GeoJSON : (1) - (2)

La structure générale du code reste semblable, nous allons juste ajouter quelques éléments.


Quelques précisions avant de commencer

Le but de cet article est de présenter une manière d'afficher un jeu de données issu de relevés GPS.

Pour pouvoir utiliser vos données sans trop de difficultés, il vous faudra :

  • Importer les données GPS dans un SIG
  • Veiller à renseigner le système de coordonnées au format latitude/longitude
  • Exporter les données au format JSON
  • Mettre votre jeu de donnée JSON dans un des répertoires de travail de votre projet.

Ajout de l'élément JQurey

Dans l'en-tête, nous ajoutons un script JQuery nous permettant de manipuler des fichers externes. 

<script src="jquery-2.1.1.min.js"></script>

Ajout du fichier GeoJSON (point)

Après avoir ajouté notre structure de base, nous utilisons la fonction getJSON pour charger notre fichier de points.

La première étape consiste à définir le chemin d'accès à la donnée : 

$.getJSON("ma_donnee_point.geojson",function(data_point){

La seconde étape va exécuter le script une fois le fichier chargé et analysé.

À l'intérieur de cette fonction, nous utilisons L.geoJson () pour recréer la couche vectorielle à partir du fichier GeoJSON. Puis en l'ajoutant à la carte avec la fonction addTo() 

L.geoJson(data).addTo(map)

Afficher toutes mes données (points + polygones)

Maintenant que nous savons afficher des fichiers GeoJSON, nous allons voir comment insérer nos deux données sur la carte.

La démarche est très simple, il suffit d'utiliser deux fois la fonction : 

$.getJSON

et penser à changer le nom de la fonction pour chaque fichier à afficher :

  • pour les polygones : 

    $.getJSON("ma_donnee_polygone.geojson",function(poly_data){
  L.geoJson(poly_data).addTo(map);
});

  • pour les points : 

    $.getJSON("ma_donnee_point.geojson",function(point_data){
  L.geoJson(point_data).addTo(map);
});
